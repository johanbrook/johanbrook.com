# Johan's site

![Build & Deploy](https://github.com/johanbrook/johanbrook.com/actions/workflows/deploy.yml/badge.svg)

## Install

Install [Lume](https://lumeland.github.io) locally:

```bash
deno run -A https://deno.land/x/lume/install.ts
```

## Build

```bash
lume
```

## Serve

```bash
lume --serve
```
